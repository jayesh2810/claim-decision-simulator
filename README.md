# Claim Decision Simulator

A simulator for claim decisions using a deterministic, auditable pipeline: eligibility, fraud, compliance, payout, and final decision. The backend is built with [FastAPI](https://fastapi.tiangolo.com/); the MVP uses explicit rules (no machine learning).

## Local development

- **Backend:** See `backend/main.py` and `requirements.txt`. Run with your preferred ASGI server (e.g. `uvicorn`).
